# ✨ Rafael Tauan — Landing Page
https://tauanapex.vercel.app/

Landing page premium desenvolvida para personal trainer, com foco em estética moderna, animações suaves e experiência visual refinada.

Projeto construído utilizando React, Vite e Tailwind CSS, priorizando performance, responsividade e componentização.

---

# 🚀 Tecnologias Utilizadas

## Front-end

* React 18
* Vite
* TypeScript
* Tailwind CSS

## UI & UX

* Embla Carousel
* Lucide React
* Motion / Framer Motion

## Ferramentas

* npm
* Git
* GitHub

---

# 📦 Instalação

```bash
npm install
```

---

# ⚙️ Scripts

```bash
npm run dev      # ambiente de desenvolvimento

```
